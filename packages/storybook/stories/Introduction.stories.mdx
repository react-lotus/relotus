# React UI Team Storybook

Storybook содержит примеры использования core-функционала проектов команды React UI Team.

Core-функционал представляет собой монорепозиторий `@relotus`.

## Группы пакетов

Пакеты контроля качества кода:

- `@relotus/tsconfig` конфигруция для typescript

Пакеты для web-проектов:

- `@relotus/utkonos` компоненты из <a href="https://www.figma.com/file/LeE5a4P9cTFQL9HIsFUtAv/UTKONOS-ВПК-UI-kit" target="_blank">UI-kit</a>

Универсальные пакеты:

- `@relotus/hooks` полезные хуки
- `@relotus/utils` полезные утилитарные функции
